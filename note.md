# 課程筆記

## 選取器

- Tag 標籤
- class 類別
- id 
- * 萬用

### 權重

    影響的元素數量越少，權重越重。(物以稀為貴)
    !important > 嵌入式 > id > class > tag > *

### CSS套用順序

    先決定權重，權重相同再看順序，後寫的優先
